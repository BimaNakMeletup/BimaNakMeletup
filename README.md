I'm a Computer Engineering student, building web/mobile systems for coursework and pulling microcontrollers apart in my free time. Most weeks split between the two — one half spent on clean architecture and APIs, the other half soldering wires and debugging why a servo won't respond.

### Right now

**Robot Pet** — a small pet robot running on ESP8266/ESP32, controlled through a web dashboard served straight off the chip. Currently reworking the UI to be more compact and properly mobile-first.

**STM ERP** — an Odoo implementation for a fictional milk tea outlet chain, built for an ERP course. Covers Inventory, HR/Payroll, POS, and Accounting, with UML docs to back it up.
